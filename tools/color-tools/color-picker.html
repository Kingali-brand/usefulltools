Color Formats
  • HEX: Hexadecimal format (#RRGGBB)
  • RGB: Red, Green, Blue values (0-255)
  • HSL: Hue, Saturation, Lightness
  • HSV: Hue, Saturation, Value
Palette Types
  • Monochromatic: Different shades of the same color
  • Analogous: Colors next to each other on the color wheel
  • Complementary: Colors opposite on the color wheel
  • Triadic: Three colors evenly spaced on the color wheel
  • Tetradic: Four colors forming a rectangle on the color wheel
Tips
  • Click on any color in the palette to use it as the main color
  • Use the copy buttons to quickly copy color codes
  • Gradients can be used in CSS for backgrounds
  • Consider color accessibility when choosing palettes
