What is Base64?

Base64 is a binary-to-text encoding scheme that represents binary data in an ASCII string format. It's commonly used to encode data for transmission over media designed to deal with textual data.

Common Uses
  • Email attachments: MIME encoding
  • Data URLs: Embedding images in HTML/CSS
  • API data: Transmitting binary data in JSON
  • Configuration: Storing binary data in text files
  • Web development: Encoding images and files
Character Set

Base64 uses 64 characters:

  • A-Z: Uppercase letters (26)
  • a-z: Lowercase letters (26)
  • 0-9: Digits (10)
  • +, /: Special characters (2)
  • =: Padding character
Important Notes
  • Base64 increases data size by approximately 33%
  • Not suitable for large files due to memory limitations
  • Base64 is encoding, not encryption
  • All processing is done locally in your browser
  • Files are not uploaded to any server
