What is URL Encoding?

URL encoding (also called percent encoding) converts special characters into a format that can be transmitted over the Internet.

Characters are replaced with a percent sign (%) followed by their hexadecimal ASCII code.

When to Use
  • Query Parameters: When URLs contain spaces or special characters
  • Form Data: When submitting forms with special characters
  • API Calls: When passing data in URL parameters
  • File Names: When URLs contain file names with spaces
Programming Examples

JavaScript:

encodeURIComponent(text)
decodeURIComponent(text)

PHP:

urlencode($text)
urldecode($text)

Python:

urllib.parse.quote(text)
urllib.parse.unquote(text)
Important Notes
  • Only encode the query string and path parts, not the entire URL
  • Don't double-encode already encoded URLs
  • Some characters like A-Z, a-z, 0-9, -, _, ., ~ don't need encoding
  • Always validate URLs after encoding/decoding
